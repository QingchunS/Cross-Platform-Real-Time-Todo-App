# React Native To‑Do App

A simple and modern Todo application built with React Native(Expo) and Convex.  
This project demonstrates full-stack development with real-time data syncing, clean UI design, and scalable backend architecture.

## Features

- ✅ Add new todos
- 🔄 Toggle todo completion
- 🗑️ Delete todos with confirmation
- ⚡ Real-time data updates (powered by Convex)
- 🎨 Clean UI with gradient styling
- 🌙 Theme support (light/dark mode)

## Setup & Installation

### 1. Clone the repo

```bash
git clone https://github.com/YOUR_USERNAME/todo-app.git
cd todo-app
```

### 2. Install dependencies

```bash
npm install
```

### 3. Set up Convex

```bash
npx convex dev
```

### 4. Start the app

```bash
npx expo start
```
